# CV 1

Currículum Vitae desarrollado como sitio web utilizando HTML y CSS.

## 📋 Descripción

Este proyecto corresponde a un currículum vitae en formato web, diseñado para ser visualizado desde cualquier navegador.

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- Visual Studio Code
- Git
- GitHub

## 📁 Estructura del proyecto

```
CV-1/
│
├── index.html
├── styles.css
├── README.md
└── assets/
```

## 💻 Cómo ejecutar

1. Descargar el proyecto.
2. Abrir la carpeta en Visual Studio Code.
3. Abrir el archivo `index.html` con Live Server o desde el navegador.
